# Memory-Website-
Online 

https://alt-magick.github.io/Memory-Website-/

Odd line numbers are questions or prompts. Even lines are scripts and answers.

It should work the same as the C++ version at:

https://github.com/alt-magick/Memory-Update


Nav:<br><br> Prev, goes to last item.<br>
Next, goes to next item.<br> 
Start, goes to beginning.<br>
Random, goes to random item.<br><br>

Actions:<br> <br>
Answer, displays answer.<br>
Clear, clears screen.<br>
Mem, flags item as memorized.<br>
Skip, toggles memorized items as ignored.<br><br>

Tools:<br><br> Jump, searches for string in questions.<br>
Fuzzy, sets percentage for correct answers.<br>
Stats, shows statistics.<br>
Load File, opens data file.<br><br>
                Swipe:<br><br>Left / Right (Next / Prev)<br> Up (Rand)<br> Down (Start)<br> Tap (Answer)<br> Hold (Mem)
